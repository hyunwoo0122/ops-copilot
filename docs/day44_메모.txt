# Day 44 — 문장 단위 Chunking + 빈 줄 제거 / 2026-05-14

## 반드시 외울 코드 패턴

### 문장 단위 chunking
def chunk_by_sentence(text: str) -> list[str]:
    lines = text.split('\n')
    chunks = []
    for line in lines:
        stripped = line.strip()    # ← strip() 문자열 반환
        if stripped:               # ← 빈 줄 제외
            chunks.append(stripped)  # ← stripped 넣기! line 아님!
    return chunks

### 빈 줄 제거
def delete_space(lines: list[str]) -> list[str]:
    result = []
    for line in lines:      # ← str 쓰지 말 것! 내장함수 덮어씌움
        if line != "":
            result.append(line)
    return result

## 절대 헷갈리면 안 되는 것 3가지

# 1. strip vs split vs splitlines
"  hello  ".strip()      → "hello"         # 문자열 ✅
"a b c".split()          → ["a","b","c"]   # 리스트 ❌ (encode 못 씀)
"a\nb\nc".splitlines()   → ["a","b","c"]   # 리스트 ❌ (encode 못 씀)

# 2. append 대상
chunks.append(line)      # ❌ 공백 포함된 원본
chunks.append(stripped)  # ✅ 공백 제거된 버전

# 3. 변수명 금지어
for str in lines   # ❌ str은 파이썬 내장함수
for line in lines  # ✅

## chunking 방식 비교 (면접에서 바로 쓸 수 있어)
| 방식        | 유사도 | 단어 잘림 | 언제 씀 |
|------------|--------|---------|--------|
| 고정 글자수  | 0.23   | 있음 ❌  | 쓰지 말 것 |
| 문장 단위   | 0.54   | 없음 ✅  | 기본 전략 |

## 시간복잡도 누적 표
| 패턴            | 복잡도      |
|----------------|-------------|
| for 루프        | O(n)        |
| set()          | O(n)        |
| 슬라이딩 윈도우  | O(n)        |
| 빈 줄 제거      | O(n)        |
| sorted()       | O(n log n)  |
| 이중 for 루프   | O(n²)       |
