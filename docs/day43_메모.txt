## 반드시 외울 코드 패턴

### chunking (슬라이딩 윈도우)
def chunk_text(text, chunk_size=100, overlap=20):
    chunks = []
    start = 0
    while start < len(text):
        chunk = text[start:start + chunk_size].strip()
        if chunk:
            chunks.append(chunk)
        start += chunk_size - overlap  # ← overlap만큼 뒤로 돌아감
    return chunks

### threshold 필터링
for score, idx in zip(distances[0], indices[0]):
    if score >= threshold:
        results.append(chunks[idx])

## 절대 헷갈리면 안 되는 것

# split() vs strip()
text.split()  → 리스트 반환 ["hello", "world"]  ← model.encode() 못 씀
text.strip()  → 문자열 반환 "hello world"       ← model.encode() 가능

# chunk 이동거리
start += chunk_size           # ❌ overlap 무시
start += chunk_size - overlap # ✅ overlap만큼 겹치면서 이동

## threshold 결정 방법
0. 일단 실행해서 실제 유사도 점수를 확인한다
1. 관련 쿼리 점수 ~ 0.23
2. 무관 쿼리 점수 ~ 0.12
3. 중간값 0.15를 threshold로 설정
→ 무조건 0.3같은 고정값 쓰지 말 것

## 시간복잡도 표 (누적)
