1️⃣ 포트폴리오 README 필수 구성
────────────────────────────────
쉽게 말하면:
  면접관이 30초 안에 읽고 "면접 불러야겠다" 고 느끼게 만드는 간판

반드시 들어가야 할 것:
  1) 한 줄 소개     → 뭘 만드는 프로젝트인지
  2) 해결하는 문제  → 왜 만드는지
  3) 숫자로 된 성과 → 정상 0.2707 / 이상 0.4874 처럼
  4) 실행 방법      → pip install ~ / python ~ 명령어
  5) 아키텍처 흐름  → 로그 → Attention → 이상점수 → 판단
  6) 일본어 요약    → 일본 기업 지원 시 필수

❌ 절대 하면 안 되는 것:
  - 숫자 없는 성과: "이상탐지 구현했습니다" → 신뢰도 0
  - 실행 방법 없음: 따라 해볼 수가 없어
  - 일본어 요약 없음: 성의 없어 보임
