## 오늘의 알고리즘 — 로그 레벨 집계

[핵심 로직]
- 딕셔너리로 레벨별 횟수 카운팅
- max(dict, key=dict.get) 으로 최다 레벨 찾기
- tuple[dict[str, int], str] 로 둘 다 반환

[배운 것 1] dict.get() 활용
- Before: if level in counts: counts[level] += 1
          else: counts[level] = 1
- After : counts[level] = counts.get(level, 0) + 1
- get(key, 기본값) = 키가 없으면 기본값 반환

[배운 것 2] 변수명으로 절대 쓰면 안 되는 것
- dict, str, list, type, id, input
- 파이썬 내장 함수를 덮어씌워서 버그 발생

[시간복잡도]
- for문 O(n) + max() O(k) = 전체 O(n)
- k는 레벨 종류 수, 최대 n이므로 O(n)이 대표

Q: 一番苦労したことは何ですか？
A: contamination=0のときIsolationForestがエラーになる
   問題がありました。max(0.01, contamination)で
   解決しました。(Day1개월차 버그 3)

Q: このコードの時間計算量は？
A: ループが一度だけなのでO(n)です。
   max()もO(k)ですが、全体はO(n)になります。
